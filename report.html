Dad Jokes & Cats Application by Zainab Ali: Project 3 Report

This application uses the icanhazdadjoke API and the Cat API to either randomly generates a dad joke or generate one from a keyword entered by the user.

Then the programs asks how the user feels after the joke, once a "mood" button is clicked, 4 randomly themed cat images are generated.

The user has the option to download the cat images that are generated.

This assignment satisfies the following project requirements:

  • Site is deployed on github.
  • Site is responsive to presentations on mobile or larger viewports.
  • Uses 2 APIs: icanhazdadjoke and the Cat API.
  • Has a search option for finding dad jokes that have certain words.
  • Displays the result of the search as the dad joke that contains the keyword(s).
  • The user is able to download the cat images that are generated.
